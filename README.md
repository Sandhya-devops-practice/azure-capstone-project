🚀 Azure Cloud Web Application Deployment (Flask + Storage + Application Gateway)

📌 Project Overview

This project demonstrates a complete cloud-based web application deployed on Microsoft Azure using a Flask backend, Azure Storage, and networking components like Application Gateway, Traffic Manager, and Virtual Network Peering.

The application supports file upload and static website hosting using Azure Blob Storage with LRS (Locally Redundant Storage) for durability and cost efficiency.

🏗️ Architecture Overview

The system is designed with scalability, availability, and security in mind.
