# 🚀 Azure Cloud Web Application Deployment (Flask + Storage + Application Gateway)

## 📌 Project Overview

This project demonstrates a complete cloud-based web application deployed on **Microsoft Azure** using a Flask backend, Azure Storage, and networking components such as **Application Gateway, Traffic Manager, and Virtual Network Peering**.

The application supports:
- File upload functionality
- Static website hosting
- Scalable cloud architecture using Azure services
- Storage using Azure Blob Storage with **LRS (Locally Redundant Storage)** for cost efficiency and durability
